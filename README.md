# Card_game

Local 1v1 deck-building card game inspired by Star Realms with a character deck twist.

## Core Idea
Players start with a weak starter deck and improve it during the match by buying new cards from a shared pool.
Unlike traditional deck-builders, cards represent items that must be equipped to a character before they can be used.

Each character has limited item slots, forcing players to make strategic decisions about which cards to use each turn.

## Gameplay
- Turn-based (1v1)
- Each turn:
  - Draw cards
  - Draw your character
  - Equip items into available slots
  - Activate equipped items
  - Buy new cards
  - Attack opponent
- Deck evolves dynamically during the game

## Features
- Local multiplayer (1v1)
- Deck-building mechanics
- Card effects system
- Expandable card pool
